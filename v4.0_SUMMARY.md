```

2. See modal with Top 3 judges
3. Each shows:
   - Rank (#1, #2, #3)

```

```
3. Click "Edit Draft"
4. See highlighted changes:
   - Yellow: Added text
   - Green: Corrected text
   - Red strikethrough: Deleted text
5. Click "Sign & Release"
6. Success: "Decision signed and released"
```
